# Changelog v0.10.0

## Features

### Agents System (#3)

- **Add expert agent definitions** for Backend, Frontend, Infrastructure, QA, and Security specialists.
- **Add multilingual support for agents documentation** — Created versions in Spanish (`README.es.md`), Basque (`README.eu.md`), Galician (`README.gl.md`), Japanese (`README.ja.md`), and English (`README.md`).
- Each document outlines the purpose, usage, and principles of the specialized agents, including installation instructions and guidelines for creating custom agents.
- **Enhance README** with detailed Agents section and usage instructions.

### CLI Enhancements (#4, #8)

- **Add agent installation options** and enhance CLI functionality for skills and agents.
- **Update language files** to include agent installation options and improve localization.
- **Enhance CLI argument parsing** for agents with corresponding tests.
- **Add CLI target configuration** and argument parsing for unattended mode.
- **Add CLI target selection messages** to localization files.
- **Enhance argument parsing** and add resolution for CLI targets.

### CLI Restructure

- **Restructure CLI codebase** for improved modularity and readability:
  - `config.js` — Centralized configuration of paths and CLI targets.
  - `discovery.js` — Skill and agent discovery functionalities.
  - `filesystem.js` — File operations related to skills and agents.
  - `interactive.js` — Separated interactive prompts for better organization.
  - `resolvers.js` — Argument parsing and resolution logic.
  - `ui.js` — Consistent logging and user feedback.
  - Removed redundant code from `install.js`, delegating responsibilities to the new modules.
